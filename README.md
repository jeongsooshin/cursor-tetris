# 테트리스 (Tetris)

순수 HTML · CSS · JavaScript로 만든 클래식 테트리스 게임입니다.  
프레임워크나 빌드 도구 없이 브라우저에서 바로 실행할 수 있습니다.

## 게임 소개

떨어지는 테트로미노(블록)를 회전하고 이동해 가로줄을 완성하면 해당 줄이 사라집니다.  
블록이 쌓여 새 블록이 등장할 공간이 없으면 게임 오버입니다.

- **7가지 블록**: I, O, T, S, Z, J, L
- **점수**: 1줄 100 · 2줄 300 · 3줄 500 · 4줄 800
- **게임 오버 후** `다시 시작` 버튼으로 처음부터 다시 플레이할 수 있습니다

## 조작법

| 키 | 동작 |
|---|---|
| `←` `→` | 좌우 이동 |
| `↓` | 소프트 드롭 (한 칸씩 빠르게 하강) |
| `↑` | 시계 방향 회전 |
| `Space` | 하드 드롭 (즉시 바닥까지 하강) |

## 로컬 실행

### 방법 1: 파일 직접 열기

`index.html`을 더블클릭해 브라우저에서 엽니다.

### 방법 2: 로컬 서버 (권장)

프로젝트 루트에서 아래 명령을 실행한 뒤, 브라우저에서 `http://localhost:8000` 으로 접속합니다.

```bash
python3 -m http.server
```

Windows에서 `python3`가 없다면 `python -m http.server`를 사용하세요.
